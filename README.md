# Sobre o projeto

O aplicativo My Weather App oferece funcionalidades avançadas para manter os usuários informados sobre as condições meteorológicas. Utilizando a geolocalização, o app identifica automaticamente a localização do usuário, fornecendo previsões precisas e atualizadas do clima para a área específica. Além disso, permite que os usuários façam pesquisas manuais para obter informações meteorológicas de outras regiões. O aplicativo também oferece diversas opções de configuração, como unidades de medida, notificações personalizadas e temas, que podem ser ajustadas e salvas diretamente no dispositivo.
